Informations générales sur le site

Rolex représente une compagnie de montre de luxe.

Caractéristiques sur le site
Couleur/images de fond Une vidéo qui joue automatiquement ainsi que plusieurs images de fond. La couleur de fond est le blanc.
Couleur du texte Blanc(sur les images et les fonds sombres), noir(sur les images de montres et les fonds blancs), vert(sur les boutons), et une couleur spéciale qui change quand on descend la page sur une des titres.
Interface de navigation Menu positionné à gauche. Il apparait quand on clique sur un bouton. Il permet de se déplacer dans plusieurs autres pages du site. Dans ce menu, on peut changer la langue du site pour un des 22 choix offerts! (anglais, français, allemand...)
Nombre d'écrans 6 écrans, avec mon écran de 21 pouces.
Présence de vidéos et d'animations Il y a une video en haut de la page d'acceuil, lorsqu'on arrive sur le site. Il n'y a pas d'animations spéciales.
Publicité Il n'y a pas de publicité externe car c'est un site qui vend des produits.
Types de contenu
Version HTML HTML5, qu'on peut remarquer par la balise au début de la page: <!DOCTYPE html>
Style de la page La page utilise la balise style directement dans le fichier HTML. On le ramarque par les balises style incluses dans la page html. Ceci est surprenant puisque ça prend beaucoup de place dans le code et ce n'est pas sastisfaisant visiblement.
Type de script Ce site utilise 4 scripts internes et 4 scripts externes. Il n'utilise pas de jQuery ni d'autre librairie.
