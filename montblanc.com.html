Informations générales sur le site

Mont-Blanc est une compagnie de produits de luxe. Leurs produits varient de stylos de luxe à des montres luxueuses.

Caractéristiques sur le site
Couleur/images de fond Le thème du site est le blanc et noir (même thème que leur logo). L'arrière-plan du site est blanc et le menu principal est noir. Il y a plusieurs images qui apportent un contrastre dans les couleurs (beige, gris, bleu...).
Couleur du texte Les textes sont majoritairement écrits en noir. Ils varient en terme d'opacité, d'épaisseur et de grosseur, dépendant de leur importance. Le texte du menu est blanc.
Interface de navigation Il y a un menu en haut complètement de la page. Il comporte 8 onglets, qui dirigent directement vers un autre partie du site. Il est simple, mais il va directement au but.
Nombre d'écrans 5 écrans, avec mon écran de 21 pouces.
Présence de vidéos et d'animations Il n'y a pas de vidéo. Il y a des petites animations lorsqu'on hover sur les icônes, mais il n'y a rien d'extravagant. Le site est simple.
Publicité Il n'y a pas de publicité externe. Cependant, Spike Lee est mis en premier plan en haut de la page, ce qu'on pourrait considérer comme une petite publicité/campagne.
Types de contenu
Version HTML HTML5, qu'on peut remarquer par la balise au début de la page: <!DOCTYPE html>
Style de la page La page utilise la balise style directement dans le fichier HTML. On le ramarque par les balises style incluses dans la page html. Ceci est surprenant puisque ça prend beaucoup de place dans le code et ce n'est pas sastisfaisant visiblement.
Types de scripts Le site utilise 2 scripts javascript internes, 1 script javascript externe et il utilise jQuery 2.2.4.
